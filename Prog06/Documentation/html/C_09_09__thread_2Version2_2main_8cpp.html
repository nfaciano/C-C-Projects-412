Detailed Description

Multithreaded Image Focusing Application (Version 2) using threads for CSC 412.

This version of the application, developed by Nicholas Faciano as of 12/3/23, is an enhancement of the initial implementation. It focuses on refining the multithreading approach for image processing, incorporating improved synchronization mechanisms. The program processes a stack of images to produce a single, sharply focused output image. Version 2 implements a more sophisticated multithreaded solution compared to Version 1, aiming for better performance and thread safety.

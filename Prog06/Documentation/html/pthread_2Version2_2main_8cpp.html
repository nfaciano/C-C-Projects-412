Detailed Description

Multithreaded Image Focusing Application for CSC 412.

This version of the application, developed by Nicholas Faciano as of 12/3/23, utilizes multithreading to process a stack of images and produce a single focused output image. The program demonstrates advanced synchronization techniques and performance optimizations in a multithreaded environment. It employs pthreads for concurrent image processing, using multiple threads to process different regions of the image stack simultaneously. This approach enhances the efficiency and speed of the image focusing process.

Key features:

  • Multithreaded processing of image stacks.
  • Region-based locking for thread synchronization.
  • Dynamic allocation of thread-specific data.
  • Implementation of pthreads for concurrent processing.
  • Utilization of atomic variables for thread-safe operations.
  • Integration with a graphical interface using OpenGL/glut for visualization.
